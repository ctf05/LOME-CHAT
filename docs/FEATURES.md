# Features

Complete feature list for LOME-CHAT, organized by category.

---

## Chat & Models

| Feature | Description |
|---------|-------------|
| Multi-Model Chat | Access GPT, Claude, Gemini, Grok, and all models available through OpenRouter from one interface. |
| Model Switching | Change to a different AI model at any point, even mid-conversation. |
| Auto Model Selection | Let the system choose the optimal model based on your task. |
| Multi-Model Conversations | Chat with multiple AI models in the same conversation to compare responses. |
| Response Regeneration | Generate a new response to the same prompt if you want a different answer. |
| Context Compacting | When approaching context limits, compress the conversation history to continue chatting. |

---

## Document Panel

A unified workspace for creating, editing, and previewing content alongside your chat.

| Feature | Description |
|---------|-------------|
| Unified Panel | Single workspace that handles code editing, document writing, and visual previews. |
| Split-Screen View | View chat and document panel simultaneously. |
| Version History | Navigate between previous versions of your documents. |
| Diff View | Compare changes between versions side-by-side. |
| Live Preview | See HTML, React, and visual content render in real-time as you edit. |
| Inline Editing | Make targeted edits to specific sections without regenerating the entire document. |
| User Editing | Directly edit documents yourself and save changes as new versions. |
| Content Publishing | Publish documents as shareable public web pages. |
